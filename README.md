# ⚡ Real-Time Collaboration Tool

A web-based real-time collaboration platform that allows multiple users to edit a shared document, chat, and view active participants simultaneously — similar to Google Docs.

---

##  Features

-  Real-time document editing
-  Live chat system
-  Active users list
-  Room-based collaboration (join using code)


---

## Tech Stack

- Frontend: HTML, CSS, JavaScript  
- Backend: Node.js, Express.js  
- Real-time Communication: Socket.io
